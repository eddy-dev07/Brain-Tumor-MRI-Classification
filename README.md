# Brain Tumor MRI Classification

## Overview
This project uses Deep Learning (CNN - EfficientNetB0) to classify brain MRI images into:

- Glioma
- Meningioma
- Pituitary
- No Tumor

## Technologies Used
- Python
- TensorFlow / Keras
- CNN (EfficientNetB0)
- OpenCV
- Streamlit
- NumPy
- Scikit-learn

## Model Details
- Input Size: 224x224
- Architecture: EfficientNetB0 (Transfer Learning)
- Activation: Softmax
- Output: 4 Classes
- Loss Function: Categorical Crossentropy
- Optimizer: Adam

## Features
- Train Model
- Evaluate Model
- Predict Single Image
- Streamlit Web App Interface
- GitHub Deployment

## How to Run
1. Activate environment
2. Install requirements
3. Run:
   streamlit run app.py
